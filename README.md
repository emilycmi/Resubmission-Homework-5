# Resubmission-Homework-5
This is a resubmission Homework 5 for SDS313 class. 
This homework looks at player data from the NWSL 
I used data from FBref.com 

RQ1: How old are professional women soccer players?
I made a histogram showing the ages of all players from my 3 teams.

RQ2: How is minutes played related to goals scored? 
I made a scatter plot showing minute player vs. goals scored
I highlighted one player, Kerry Abello

How to run? 
First, you open the R script file in RStudio.
Make sure to have NWSl.csv downloaded in your working directory. 
Make sure ggplot2 is installed. 
set your R studio working directory to the directory that the NWSL file is in.
run the script 
The graphs will show up in the plots section
